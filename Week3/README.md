
# Week 3 - Customer Intelligence System using Classification, Ensemble Learning and Clustering

## Objective

Develop an intelligence system using socio-economic and health indicators to identify countries requiring aid using machine learning techniques.

## Dataset

- Dataset Name: Country-data.csv
- Total Countries: 167
- Features Used:
  - child_mort
  - exports
  - health
  - imports
  - income
  - inflation
  - life_expec
  - total_fer
  - gdpp

## Techniques Used

### Clustering
- KMeans Clustering
- DBSCAN

### Classification & Ensemble Learning
- Random Forest
- XGBoost

## Project Workflow

1. Data Understanding  
2. Exploratory Data Analysis (EDA)  
3. Feature Scaling  
4. KMeans Clustering  
5. DBSCAN Clustering  
6. Classification using Ensemble Models  
7. Model Evaluation  
8. Feature Importance Analysis  
9. NGO Recommendation  
10. Final Insights  

## Results

- Cluster 1 showed higher child mortality and lower income levels.
- Cluster 0 showed higher income and GDP values.
- High-risk countries were identified for aid prioritization.
- Machine learning improved segmentation and decision-making.

## Business Impact

This system helps HELP International prioritize countries for aid allocation using socio-economic indicators and machine learning-based segmentation.
